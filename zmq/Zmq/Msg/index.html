val init_data : ?⁠offset:int ‑> ?⁠length:int ‑> bigstring ‑> t

Initialize a new message with the given data. The data will be kept alive for the lifetime of the message.

  • Parameter offset: specifies an offset from the start of the given block to use. Defaults to 0.
  • Parameter length: specifies the number of bytes, starting from offset, to use. Defaults the length of data - offset.
